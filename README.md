# kfi-fhir-input
This model represents the schema target for release into FHIR. Classes have 
been defined and annotated to make building FHIR ETL smoother.

## Official site
[Official Model Website](https://carrollaboratory.github.io/kfi-fhir-input/)

## Notes for resusing release software
For other models which may require the autogeneration of SQL Alchmey files that
can be added as dependencies to your python tooling, see the notes for 
[reuse.](https://carrollaboratory.github.io/kfi-fhir-input/)
